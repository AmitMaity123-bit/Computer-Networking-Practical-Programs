# Dijkstra Algorithm

## Objective

To find the shortest path from a source node to all other nodes in a network.

## Concepts Used

* Graph Theory
* Link State Routing
* Shortest Path Algorithm

## Features

* Accepts adjacency matrix
* Calculates shortest path
* Displays minimum distance

## Applications

* GPS Navigation
* Network Routing
* Traffic Management

## Algorithm

1. Select source node
2. Mark source distance as 0
3. Select nearest unvisited node
4. Update neighboring distances
5. Repeat until all nodes visited
